1. 补上了 Discovery。
- 现在不需要先知道对方 DID 才能连接，而是能先按能力检索候选 Agent。

- 现在不是只有认证成功/失败，而是失败后能沉淀证据、上链举报、执行治理动作。

3. 补上了 A2A / MCP 兼容。
- 让 Agent 不只是“会认证”，还可以用标准化路径互通和调工具。

4. 补上了统一实验框架。
- 现在已经不是零散 demo，而是可以一条命令跑全流程并自动出报告。
